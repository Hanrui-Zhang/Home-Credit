# Home Credit Portfolio

## Overview
This repository presents my individual work for the Home Credit Default Risk project. It demonstrates my data analysis, feature engineering, and modeling process using real-world financial data.

---

## Business Problem and Project Objective
Home Credit aims to provide loans to individuals with limited credit history. The key challenge is identifying which applicants are likely to repay loans and which may default.

The objective of this project is to build a predictive model that helps improve lending decisions and reduce default risk.

---

## Our Group's Solution
Our group approached this problem using machine learning techniques. We explored the dataset, cleaned and transformed variables, engineered useful features, and built predictive models to estimate default probability.
